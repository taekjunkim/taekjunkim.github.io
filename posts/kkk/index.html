Link_nwb_file_demo

General information¶

  • This code shows how to load/read V4-PFC dual recording data collected from monkey L left hemisphere

    • Two NPX probes were used to simultaneously record the activity of V4 and PFC neurons
    • To examine the influence of PFC feedback on V4, cortical cooling was applied to reversibly inactivate PFC

  • Monkey performed 2-AFC shape discrimination task

    • match: when the reference stim is the same as the test stim
    • non-match: when the reference stim and test stim are different

  • Trial sequence

    1. Reference stim appeared at the center of the screen for 400 ms
    2. After 200 ms blank period,
    3. Test stim appeared and moved behind the occluder (>400 ms)
      • To titrate the task difficulty, we set different occluder levels.
    4. Monkey makes a saccadic eye movement to report his discimination

To compute spike density function¶

In [129]:
def getSDF(spkTrain,FS=1000,AllorMean=1):
    """
    FS: sampling rate 
    AllorMean = 1 to get SDFs for all trials
    AllorMean = 0 to get the averaged SDF across all trials
    """

    # Make gaussian kernel window
    sigma = 5; 
    t = np.arange(-3*sigma,3*sigma+1); 

    y = (1/sigma*np.sqrt(np.pi*2)) * np.exp(-(t**2)/(2*sigma**2)); 
    window = y[:]; 
    window = window/np.sum(window); 

    # convolution
    sdf = np.zeros(np.shape(spkTrain)); 
    for i in np.arange(np.shape(spkTrain)[0]):
        convspike = np.convolve(spkTrain[i,:],window); 
        pStart = int(np.floor(len(window)/2)); 
        pEnd = int(np.floor(len(window)/2)+np.shape(spkTrain)[1]); 
        convspike = convspike[pStart:pEnd]; 
        
        convspike[:pStart] = convspike[pStart]; 
        convspike[-pStart:] = convspike[-pStart]; 
        sdf[i,:] = convspike; 
    sdf = sdf*FS; 

    if AllorMean==0:
        sdf = np.mean(sdf,axis=0); 
    
    return sdf;

Columns in trials table¶

  • session: session ID from session_00 to session_26
  • trialNum: each session has different numbers of trials (start from 1)
  • ref_stimID: reference stimID (presented at the center of the screen)
  • test_stimID: test stimID (presented at V4 cells' RF)
  • pdOnTS_V4: in each trial, there are two stim onset times (sec), each of which is for ref_stim and test_stim
    • V4 spike times from the same session could be associated with these pdOnTS values
    • You don't need to align spike times to pdOnTS for yourself. It is already done and could be found in units table (see below)
  • pdOffTS_V4: in each trial, there are two stim offset times (sec), each of which is for ref_stim and test_stim
    • V4 spike times from the same session could be associated with these pdOffTS values
  • pdOnTS_PFC: for PFC, because V4 and PFC raw data were saved in different files
    • PFC spike times from the same session could be associated with these pdOnTS values
    • You don't need to align spike times to pdOnTS for yourself. It is already done and could be found in units table (see below)
  • pdOffTS_PFC: for PFC
    • PFC spike times from the same session could be associated with these pdOffTS values
  • eyeH: two second duration horizontal eye signal (-1 to 1 seconds from test stim onset)
  • eyeV: two second duration vertical eye signal (-1 to 1 seconds from test stim onset)
  • correct: monkey performance: correct (1), incorrect (0)
  • pypeRT: ignore this
  • occl_width: Actually, this refers to slit_width. So, please be careful with interpretation
    • 0 for no occluder. If occl_width > 0, the bigger values mean more shape parts are visible
    • Or simply, just think that 0 for non-occluded condition, non-zero for occluded condition
  • dir: moving direction of the test stim. In most sessions, two opposite directions were used.
  • cooling_temp: PFC cooling temperature. This is useful to segerate 'cool' and 'warm' trials.

Columns in units table¶

  • session: session ID from session_00 to session_26
    • To view trial information for a specific unit, you’ll need to retrieve data from the trials table by matching the session ID.
  • hemisphere: LH (left) or RH (right)
  • unitID: unit ID in a session.
    • In each session, unit ID doesn't always start from 0, as many multi-unit clusters were excluded in this dataset
  • location: V4 or PFC
  • depth: the distance from the probe tip
    • You might expect that units near the probe tip would be located in deeper layers
    • but the actual positioning depends on factors like penetration angles and the shape of gyrus folding, which we don't have information on.
  • avgWF: averaged waveform shape
  • normWF: normalized waveform shape
  • spike_times: spike times of a specific unit collected from all trials
    • If a unit is from V4 and you want to align it to the test stimulus onset time, you should reference pdOnTS_V4[1] (the second value of pdOnTS_V4) in the trials table
  • spkMtx: (N_trials x N_timepoints), where N_timepoints = 2000 ms
    • This is spike times aligned to the test stimulus onset time (from -1000 to 1000 ms with respect to test stim onset time)
    • Rows in spkMtx are matached to rows in trials table with the same session ID

How to see cortical temperature changes in a session¶

  • The code below shows how to see cortical temperature changes from all recording sessions
  • Most sessions included multiple warm-cool cycles.
  • You can distinguish between warm and cool trials by applying a threshold (in this case, I used 20)
  • Cooling was not well controlled in session_06 and session_07

Compute Temp-Resp correlation¶

In [145]:
temp_resp_corr = np.empty((0,2));  # two columns for rval, pval
N_units = np.shape(units)[0]; 

for u in tqdm(np.arange(N_units)):
    session_ID = units.at[u,'session']; 

    session_trials = trials[trials['session']==session_ID]; 
    session_temp = session_trials['cooling_temp'].values; 

    unit_spkMtx = np.array(units.at[u,'spkMtx']).astype(float); 
    session_resp = np.sum(unit_spkMtx[:,1000:1500],axis=1);     # spike counts from 0 to 500 (I didn't include the later time points, because monkeys reation time varied)

    [rval,pval] = stats.pearsonr(session_temp, session_resp);     
    result_array = np.array([rval, pval]).reshape(1,2); 
    temp_resp_corr = np.concatenate((temp_resp_corr, result_array)); 

### columns added to units table
units['T-R_rval'] = temp_resp_corr[:,0]; 
units['T-R_pval'] = temp_resp_corr[:,1];
